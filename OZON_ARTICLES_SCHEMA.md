# Схема системы артикулов Ozon

## 🏗️ Архитектура системы

```
┌─────────────────────────────────────────────────────────────────┐
│                         OZON API                                 │
│  (Товары с offer_id, названиями, описаниями, ценами)           │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│              ozon-article-mapper.ts                              │
│  • detectCategoryFromName() - определение категории              │
│  • transformOzonToProduct() - трансформация товара               │
│  • transformOzonProducts() - массовая трансформация              │
│  • OZON_TO_ARTICLE_MAP - маппинг offer_id → артикул             │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│            ozon-article-template.ts                              │
│  • ARTICLE_PREFIXES - префиксы категорий                         │
│  • generateArticle() - генерация артикула                        │
│  • getNextArticle() - следующий доступный                        │
│  • parseArticle() - парсинг артикула                             │
│  • isValidArticle() - валидация                                  │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                    Product (внутренний формат)                   │
│  {                                                               │
│    id: "CLO-DRS-001",                                           │
│    slug: "clo-drs-001",                                         │
│    name: "Платье макраме бохо белое",                           │
│    category: "clothes",                                          │
│    subcategory: "dresses",                                       │
│    ozonId: 12345,                                               │
│    ...                                                           │
│  }                                                               │
└─────────────────────────────────────────────────────────────────┘
```

## 📊 Структура артикула

```
┌─────────────────────────────────────────────────────────┐
│                    CLO-DRS-001                          │
│                     │   │   │                           │
│                     │   │   └─── Номер (3 цифры)       │
│                     │   └─────── Подкатегория (3 буквы) │
│                     └─────────── Категория (3 буквы)    │
└─────────────────────────────────────────────────────────┘
```

## 🗂️ Иерархия категорий

```
Stariva Макраме
│
├── 🧥 Одежда (CLO)
│   ├── Платья (DRS)      → CLO-DRS-XXX
│   ├── Топы (TOP)        → CLO-TOP-XXX
│   └── Пояса (BLT)       → CLO-BLT-XXX
│
├── 🏠 Интерьер (INT)
│   ├── Абажуры (LMP)     → INT-LMP-XXX
│   ├── Типи (TIP)        → INT-TIP-XXX
│   ├── Панно (PAN)       → INT-PAN-XXX
│   ├── Салфетки (PLM)    → INT-PLM-XXX
│   └── Кашпо (PLN)       → INT-PLN-XXX
│
└── 👜 Сумки (BAG)
    ├── Шопперы (TOT)     → BAG-TOT-XXX
    ├── Через плечо (CRS) → BAG-CRS-XXX
    └── Корзины (BSK)     → BAG-BSK-XXX
```

## 🔐 Правила валидации

```
✓ Валидные артикулы:
  CLO-DRS-001  ← Правильный формат
  INT-LMP-999  ← Максимальный номер
  BAG-TOT-042  ← С ведущими нулями

✗ Невалидные артикулы:
  CLO-DRS-1    ← Не 3 цифры
  clo-drs-001  ← Строчные буквы
  CLODRS001    ← Без дефисов
  CLO-DRESS-1  ← Неправильный код подкатегории
  XXX-YYY-001  ← Несуществующая категория
```

## 🚀 Workflow использования

```
1. Новый товар на Ozon
   └─→ API возвращает данные

2. Вызов transformOzonProducts()
   └─→ Определение категории
       └─→ Генерация/получение артикула
           └─→ Извлечение данных
               └─→ Создание Product

3. Product готов к использованию
   └─→ Отображение в каталоге
       └─→ Связь с Ozon через ozonId
```

Эта схема показывает полную архитектуру системы артикулов для товаров Ozon в проекте Stariva.
