## 🎯 Формат артикула

```
XXX-YYY-NNN
│   │   └─── Номер (001-999)
│   └─────── Подкатегория (3 буквы)
└─────────── Категория (3 буквы)
```

## 📋 Все категории (11 подкатегорий)

### 🧥 Одежда (3)
- `CLO-DRS` - Платья
- `CLO-TOP` - Топы
- `CLO-BLT` - Пояса

### 🏠 Интерьер (5)
- `INT-LMP` - Абажуры
- `INT-TIP` - Типи
- `INT-PAN` - Панно
- `INT-PLM` - Салфетки
- `INT-PLN` - Кашпо

### 👜 Сумки (3)
- `BAG-TOT` - Шопперы
- `BAG-CRS` - Через плечо
- `BAG-BSK` - Корзины

## 💡 Примеры использования

### Создать артикул
```typescript
import { generateArticle } from '@/data/ozon-article-template';
const article = generateArticle('clothes', 'dresses', 1); // CLO-DRS-001
```

### Трансформировать товары Ozon
```typescript
import { transformOzonProducts } from '@/lib/ozon-article-mapper';
const products = transformOzonProducts(ozonApiData.result.items);
```

### Получить следующий артикул
```typescript
import { getNextArticle } from '@/data/ozon-article-template';
const next = getNextArticle('interior', 'lampshades', existing); // INT-LMP-003
```
